## 📌 О проекте

Одностраничный сайт для VR-клуба PARADOX в Новосибирске. Сайт выполнен в стиле киберпанк с неоновой эстетикой, адаптивным дизайном и интерактивными элементами.

### Особенности:
- 🎨 Современный киберпанк-дизайн с неоновыми эффектами
- 📱 Полностью адаптивная верстка (мобильные, планшеты, десктоп)
- 🎮 Фильтрация игр по категориям (VR, PS5, MOZA)
- 🎁 Электронные подарочные сертификаты (PDF)
- 🎂 Приглашения на день рождения (PDF)
- 💬 Интерактивное модальное окно бронирования
- 📍 Интеграция с Яндекс.Картами
- ⭐ Ссылки на отзывы в Яндекс.Картах и 2ГИС
- 📊 SEO-оптимизация и микроразметка Schema.org
